Xafari — это расширение и обогащение DevExpress XAF (eXpressApp Framework), 
ориентированное на разработку бизнес-ориентированных приложений на .NET/C#.
Он предоставляет:
- множество дополнительных модулей и компонентов
- расширенные инструменты UI
- бизнес-процессы, безопасность, обработку данных
- поддержку WinForms, WebForms, ASP.NET MVC и др

Документация: https://documentation.galaktika-soft.com/xafari/
Официальные сайты: https://galaktika.ru/xafari и https://galaktika-soft.com/xafari
Демо: https://demos.galaktika-soft.com/xafari/web/FeatureCenter/#ViewID=Welcome_Dashboard

Сайт официальной документации Xafari — правильный источник знаний для проекта.
MCP подходит как умный слой поверх этого сайта, чтобы получить:
- быстрые ответы
- примеры кода
- контекстные объяснения

Для разработки в Cursor с использованием этого фреймворка нужно создать MCP-агента
- Использовать сайт документации как основной источник знаний
- Организовать MCP-сервер поверх этой документации

MCP делать так, чтобы он:
- индексировал HTML страниц сайта
- умел искать нужные разделы
- обрабатывал запросы вроде:
  - «Что делает компонент X?»
  - «Покажи пример настройки модуля Business Components»
  - «Как работает модуль Performance Enhancement?»
  - «Где описано подключение к DevExpress XAF?»

Обязательные методы:
search_docs(query: string) — ищет по тексту сайта
get_page(slug: string) — возвращает содержание раздела
get_examples(topic: string) — извлекает примеры по теме
explain_concept(name: string) — объясняет конкретный концепт

Детали реализации: https://chatgpt.com/share/697e6620-1244-8000-94b4-c621156563ab
